# Exec Plan Maintenance

Keep active plans short, decision-complete, and current.

## Workflow

1. Create or update a plan in `docs/exec-plans/active/` for multi-step work.
2. Keep the plan aligned with the actual code and docs, not with stale intentions.
3. Record what counts as done in `## Success Criteria`.
4. Move the file to `docs/exec-plans/completed/` when the work is complete.
5. If scope changes materially, revise the same plan instead of scattering notes.
